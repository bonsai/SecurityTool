# SecurityTool v2.0 (掃除夫 木村さん)

Windows 10向けの常駐型PCクリーナーアプリケーションです。

PowerShellスクリプト版 `v1.x` からGo言語で再実装され、システムトレイに常駐して定期的にPCのお掃除をサポートする「掃除夫 木村さん」として生まれ変わりました。

![screenshot](https://user-images.githubusercontent.com/12345/67890.png) <!-- ダミー画像 -->

## 主な機能

- **常駐監視**: システムトレイに常駐し、バックグラウンドで動作します。
- **定期スキャン**: 3時間ごとにクリーンアップ可能な項目を自動で調査します。
- **調査結果の通知**: スキャン完了後、削除可能な項目と概算サイズをデスクトップ通知でお知らせします。
- **ユーザー確認**: 実際の削除は行わず、ユーザーが確認した上で手動実行する安全設計です。
- **手動実行**: トレイアイコンのメニューから、好きなタイミングで調査やクリーンアップを実行できます。
- **ログ記録**: すべての動作は `kimura-san_log.txt` に記録され、いつでも確認できます。

## クリーンアップ対象

現在、以下の項目に対応しています。

- **ジャンクファイル**: Windowsの一時ファイル (`%TEMP%`) など。
- **Docker**: 未使用のコンテナ、イメージ、ネットワーク、ボリューム (`docker system prune` を実行)。

## 使い方

1.  リリースページから `kimura-san.exe` と `assets` フォルダをダウンロードします。
2.  `kimura-san.exe` と `assets` フォルダを同じディレクトリに配置します。
3.  `kimura-san.exe` を実行します。
4.  システムトレイに「掃除夫 木村さん」のアイコンが表示されます。

## ビルド方法

```bash
# 依存関係の取得
go mod tidy

# Windows向けにビルド
GOOS=windows GOARCH=amd64 go build -ldflags="-H windowsgui" -o kimura-san.exe ./cmd/kimura-san/
```

`-ldflags="-H windowsgui"` を付けることで、実行時にコマンドプロンプトが表示されないようになります。
