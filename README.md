# SecurityTool

Windows向けのシステムメンテナンスツール集です。

## バージョン

- **[v2: 掃除夫 木村さん (Go言語版)](./v2/README.md)**
  - Windows 10向けの**常駐型**PCクリーナーアプリケーションです。
  - システムトレイに常駐し、定期的にPCのお掃除をサポートします。
  - **現在開発中の最新版です。**

- **[v1: PowerShellスクリプト版](./v1/README.md)**
  - C++ DLLをコアエンジンとし、PowerShellから呼び出す従来版です。
  - ジャンクファイル、レジストリ、Docker、MLキャッシュ等のクリーンアップに対応しています。
